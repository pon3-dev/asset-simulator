# 資産取り崩しシミュレーター

モンテカルロ法による資産取り崩し（リタイアメント）シミュレーター。単一のHTMLファイルのみで動作し、ビルドやサーバーは不要です。

## 使い方

### ローカルで開く

`index.html` をブラウザで開くだけです（ダブルクリックでOK）。

### GitHub Pagesで公開する

https://pon3-dev.github.io/asset-simulator/ で公開されています。

## 何をするツールか

株式・投資信託・現金・貸付などの現在の資産と、収入・生活費のフェーズ（本人の年齢で区切りを自由に追加可能）、年金の開始年齢・金額を入力すると、目標年齢まで資産が持つ確率をモンテカルロ・シミュレーション（既定10,000試行）で算出します。

- 株式・投資信託は幾何ブラウン運動（対数正規分布）でシミュレートし、実質リターン（インフレ調整後）で計算します。入力する金額はすべて現在価値（実質）ベースです。
- 現金は「保持上限」まで手元に残し、超過分は投資へ、不足分は投資資産の取り崩しで補います。
- 資産が一度ゼロになった試行は、それ以降も回復せずゼロのまま推移します（破綻は不可逆な状態として扱う）。
- 子の生年はカンマ区切りで複数入力でき、小卒・中卒・高卒・大卒・院卒の年齢の目安が表示されます（フェーズの区切りの参考用で、シミュレーションには自動反映されません）。
- 収入・生活費のフェーズは「終了年齢（本人）」を指定して好きなだけ追加でき、空欄のフェーズは目標年齢まで続きます。

## データの保存について

入力した数値はファイルやリポジトリには一切保存されません。ブラウザの `localStorage` にのみ保存され、次回同じブラウザで開いたときに自動復元されます。他の端末やブラウザとは共有されず、リポジトリを公開しても入力内容が外部に漏れることはありません。
